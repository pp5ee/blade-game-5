# 🔪 转刀刀游戏 (Knife Throwing Game)

一个基于HTML5 Canvas开发的像素风格动作游戏，玩家需要收集刀提升攻击力，击败所有NPC敌人来获得胜利。

## 🎮 游戏特色

- **像素艺术风格** - 复古的视觉效果
- **智能NPC AI** - 四种行为状态：游荡、追逐、攻击、逃跑
- **刀收集机制** - 红、黄、蓝三种刀类型，提供不同的攻击力加成
- **随机地图生成** - 每次游戏都有新的地图体验
- **完整的战斗系统** - 包含暴击、伤害加成等机制
- **性能优化** - 动态质量调整确保流畅体验

## 🚀 快速开始

### 运行游戏

1. 克隆或下载项目文件
2. 使用现代浏览器打开 `index.html`
3. 点击"开始游戏"按钮开始游戏

### 游戏控制

- **移动控制**: WASD 或 方向键
- **暂停/继续**: P 键
- **重新开始**: R 键
- **调试模式**: Ctrl + D

## 🎯 游戏目标

1. 收集地图上的刀来提升攻击力
   - 红刀: +15% 攻击力
   - 黄刀: +25% 攻击力  
   - 蓝刀: +35% 攻击力

2. 击败所有NPC敌人
   - 普通NPC: 中等生命值和攻击力
   - 快速NPC: 低生命值，高移动速度
   - 强力NPC: 高生命值，高攻击力

3. 注意保护自己的生命值，避免被NPC击败

```
src/

├── game/             # 游戏核心

├── systems/          # 游戏系统

├── map/              # 地图相关
```

## 🛠️ 技术实现

### 核心技术
- **HTML5 Canvas** - 游戏渲染引擎
- **原生JavaScript** - 游戏逻辑实现
- **模块化架构** - ES6模块系统
- **面向对象设计** - 清晰的类结构

### 游戏系统
- **游戏循环** - 基于requestAnimationFrame的高效循环
- **碰撞检测** - 矩形碰撞检测算法
- **AI系统** - 状态机驱动的NPC行为
- **地图生成** - 基于网格的随机地图生成
- **性能优化** - 动态质量调整和资源管理

## 🎨 游戏设计

### 角色设计
- **玩家**: 绿色方块，可移动，收集刀提升攻击力
- **NPC**: 三种类型，不同颜色和属性
- **刀**: 三种类型，不同颜色和加成效果

### 视觉效果
- 像素风格的游戏元素
- 动态的UI显示
- 受击和收集效果

### 音效系统
- 收集刀的音效
- 战斗音效
- 背景音乐（预留接口）

## 🔧 开发指南

### 扩展游戏

1. **添加新NPC类型**:
   ```javascript
   // 在NPC.js中添加新类型
   case 'special':
       this.maxHp = 80;
       this.speed = 2.5;
       this.damage = 10;
       break;
   ```

2. **添加新刀类型**:
   ```javascript
   // 在Knife.js中添加新类型
   purple: {
       damageBonus: 0.45,
       color: '#8a2be2',
       rarity: 0.05,
       name: '紫刀'
   }
   ```

3. **修改游戏平衡**:
   - 调整`CombatSystem.js`中的伤害参数
   - 修改`MapGenerator.js`中的地图生成参数
   - 调整`Game.js`中的游戏配置

### 性能优化

游戏已内置性能优化功能：
- 动态质量调整（高/中/低）
- 实体数量限制
- 内存使用监控
- 资源自动清理

## 📊 游戏平衡

### 伤害计算
```
总伤害 = 基础伤害 × (1 + 刀加成) × 暴击倍率
```

### NPC生成概率
- 普通NPC: 60%
- 快速NPC: 30%  
- 强力NPC: 10%

### 刀生成概率
- 红刀: 60%
- 黄刀: 30%
- 蓝刀: 10%
