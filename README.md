# 转刀刀 - 像素风格刀战游戏

转刀刀是一款基于HTML5 Canvas的单机网页游戏，玩家控制角色在随机生成的地图上收集不同颜色的刀，与主动攻击的NPC进行战斗。

## 🎮 游戏特色

- **随机地图生成**：每次游戏都有不同的地图布局
- **三种颜色刀系统**：红、黄、蓝三色刀，红色最厉害（红=黄×2=蓝×4）
- **主动NPC AI**：NPC会主动追踪并攻击玩家
- **像素风格界面**：复古像素艺术风格的游戏界面
- **动态难度系统**：根据玩家收集的刀数量动态调整游戏难度
- **性能优化**：智能NPC管理和渲染优化

## 🎯 游戏玩法

### 基本操作
- **移动**：使用方向键或WASD键移动角色
- **收集刀**：移动到刀的位置自动收集
- **战斗**：与NPC相遇时自动进行战斗

### 游戏规则
1. **刀收集**：收集红色、黄色、蓝色三种刀
2. **伤害计算**：红色刀伤害最高（红=黄×2=蓝×4）
3. **战斗机制**：基于刀数量和颜色计算伤害
4. **胜利条件**：击败所有NPC获得胜利
5. **失败条件**：被NPC击败游戏结束

## 🛠️ 技术架构

### 核心技术栈
- **HTML5 Canvas**：游戏渲染引擎
- **原生JavaScript**：游戏逻辑实现
- **CSS3**：像素风格界面设计
- **Jest**：单元测试框架

### 核心模块

#### 1. 游戏主循环 (Game.js)
- 管理游戏状态和实体
- 处理游戏循环和渲染
- 协调各个系统间的交互

#### 2. 玩家控制 (Player.js)
- 键盘输入处理
- 移动控制和碰撞检测
- 刀收集和状态管理

#### 3. NPC AI (NPC.js)
- 主动追踪玩家行为
- 智能移动决策
- 战斗交互逻辑

#### 4. 地图生成 (MapGenerator.js)
- 随机地图生成算法
- 障碍物布局优化
- 地图连通性检查

#### 5. 战斗系统 (CombatSystem.js)
- 伤害计算公式
- 暴击和防御机制
- 战斗结果解析

#### 6. 性能优化 (OptimizationManager.js)
- 动态NPC数量控制
- 脏矩形渲染优化
- 空间分割碰撞检测

## 🎨 游戏设计

### 像素风格实现
游戏采用复古像素艺术风格，通过CSS实现：
- 像素边框和阴影效果
- 复古字体和颜色搭配
- 响应式布局设计

### 伤害计算公式
```
总攻击力 = (红刀数量 × 4) + (黄刀数量 × 2) + (蓝刀数量 × 1)
```

### 动态难度系统
- NPC数量随玩家刀数量增加
- 性能自适应优化
- 智能资源管理

## 🔧 配置选项

游戏支持多种配置选项：

### 地图配置
```javascript
mapSize: { width: 20, height: 15 }  // 地图尺寸
cellSize: 40                        // 网格大小
obstacleDensity: 0.2                // 障碍物密度
```

### 刀生成配置
```javascript
knifeProbabilities: {
    red: 0.1,    // 红色刀概率
    yellow: 0.3, // 黄色刀概率
    blue: 0.6    // 蓝色刀概率
}
```

### 难度配置
```javascript
dynamicDifficulty: {
    enabled: true,
    minNPCs: 3,
    maxNPCs: 8,
    difficultyScale: 0.1
}
```
